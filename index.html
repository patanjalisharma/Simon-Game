Simon Game

Press any key to start the game

    ## How to Play Simon Game
  1. Start - Press any key on keyboard to begin. Simon will flash a color.
  2. Watch - Memorize the color sequence as it grows each round.
  3. Repeat- Click the colors in the same order.
  4. Advance - Complete the sequence to move to the next round.
  5. Game Over - If you click incorrectly, the game resets.
